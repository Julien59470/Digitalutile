# Digitalutile — Maquette espace client

Maquette statique interactive et responsive d’un espace client / administrateur Digitalutile.

## Direction UI/UX

La maquette reprend directement le langage visuel du site Digitalutile :

- fond très clair `#fbfbfc` ;
- surfaces blanches et bordures fines ;
- ombres légères ;
- accents violet–magenta ;
- boutons sombres ;
- badges arrondis et typographie ample ;
- navigation desktop, tablette et mobile.

## Vues disponibles

- tableau de bord client ;
- demandes et fiches contacts ;
- campagnes et budgets ;
- journal des actions ;
- documents ;
- paramètres ;
- vue administrateur multi-clients.

La maquette fonctionne sans compilation et peut être hébergée directement avec GitHub Pages.

Toutes les données affichées sont fictives et servent uniquement à la démonstration.
