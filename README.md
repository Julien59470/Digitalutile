# Digitalutile — Dashboard Client & Admin

Maquette front-end interactive réalisée pour illustrer une proposition de dashboard client et administrateur pour Digitalutile.

## Fonctionnalités présentées

- Dashboard client : score de visibilité, statistiques Google Ads, Google Business Profile, leads, journal d’actions, calendrier, benchmark concurrentiel, heatmap et rapports PDF.
- Dashboard administrateur : portefeuille clients, campagnes, leads consolidés, priorités équipe et intégrations.
- Navigation interactive entre les vues client et administrateur.
- Interface responsive pour ordinateur, tablette et mobile.

Toutes les données affichées sont fictives et servent uniquement à la démonstration.

## Stack

HTML, CSS et JavaScript natifs. Aucun build ni dépendance applicative n’est nécessaire.
